bus的简单使用(非父子组件之间的传递)

1.在main.js中注册全局的bus Vue.prototype.bus=new Vue(); //可先将new Vue封装到一个工具文件bus.js里，再调用引入 2.一个组件触发事件: this.bus.$emit('自定义事件名',data) methods:{ handleClicks(){ this.bus.$emit('open-xx',true) } } 3.一个组件监听 this.bus.$on("自定义事件名", msg => {}) ，自定义事件名可实现多个监听 mounted() { //在实例构建好以后监听自定义事件open-xx this.bus.$on("open-xx", msg => { this.show = msg; }); }
