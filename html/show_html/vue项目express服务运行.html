vue项目打包dist在express上运行

//全局安装（-g）express-generator npm install express-generator -g //新建项目（项目名server） express -e server 命令行末出现三条指令: cd xx npm install npm start //运行express服务 //在浏览器打开 http://localhost:3000/ ，默认端口号3000 此时出现 welcome to express //将vue项目打包 得到dist文件夹 将dist文件夹下所有文件复制到express项目的public中 //注释掉 express项目中app.js 内的 app.use('/', indexRouter); //执行npm start 访问http://localhost:3000/
