bootstrap

类名 添加对象 作用
container 最外围盒子 提供响应式盒子
dl-horizontal dl 使dl下的dt和dd成水平整齐排列
list-inline ul 将li放置在同一行，并添加少量padding
table table 使table规范化
table-striped table 使tbody内容呈现斑马条纹
table-bordered table 为单元格添加边框
table-hover table tbody每一行悬停响应
table-condensed table 让表格更加紧凑
success,info,warning,danger tr，td 为对应行或单元格添加四种不同的标注颜色
table-responsive .table的父盒子 包裹.table使其变为响应式table，小屏幕水平滚动
text-lowercase p,span等 使文本变为小写字母
text-uppercase - 使文本变为大写字母
text-capitalize - 使文本每个单词首字母大写
form-control input/textarea/select width:100%,形成特殊控件与label组合
form-group div等 用于包裹label+form-control获得更好的排列
form-inline form 使元素(form-group等)排列在同一行，可能需要手动设置padding等
input-group-addon div等 为控件前后添加小耳朵获得更好的体验
form-horizontal form 使form-group中的label和控件水平排列，注册登陆常用
row 栅格系统父盒子 用于包裹col-xx-xx类型的栅格元素，有时也可不用包裹
col-xs-，col-sm-，col-md-，col-lg- 需要调节栅格占比的元素上 搭配占比数字获得多样的排列，分别适用于小屏，平板，桌面显示器，大屏幕显示器，pc端常用md
col-md-1/-2/-3/-4..12/ 需要调节栅格占比的元素上 整个宽度为12份，按占比数字获得对应宽度，超出后排在下一行，元素组合排列获得更好的效果，xs、sm、lg同理
col-md-offset-* 需要调节栅格占比的元素上 将元素左侧空出对应占比宽度，比如对齐按钮时
checkbox，redio div等 用于包裹checkbox、redio控件及其文字说明，控件及文字一般包裹在一个label中
disabled checkbox、redio盒子 使整个控件以及文字呈现不可选中的状态，控件需要设置disabled属性
checkbox-inline、radio-inline 包裹控件的元素div或者直接label上 使控件排列在一行上
form-control-static 纯文本的盒子 纯文本信息需要和label显示在同一行，显示在原本控件的位置上
has-success/warning/error 包裹label和控件的盒子（form-group） 控件框和文字呈现对应的颜色，常用于输入提示
input-lg/sm input/select 调节控件高度，宽度还是由col-xx-*调节
form-group-lg/sm 表单组合 同时调节文字与控件高度
btn a/button/input(button)/submit 提供按钮样式
btn-default/primary/success/info/warning/danger/link .btn元素 为btn添加一些特殊色彩样式
btn-lg/sm/xs/block .btn元素 提供大/小/超小尺寸/100%宽度的按钮
img-responsive img 图片支持响应式布局
close button 构建一个关闭图标，button内放一个'span & times；/span'
pull-left/right 需要浮动的元素 使元素左右浮动float
clearfix 清除浮动的父盒子 为父元素添加以清除浮动
show/hidden 显示或隐藏的元素 显示、隐藏display：
invisible 需要隐身的元素 隐藏元素visibility：hidden
text-overflow 文本需要截断隐藏的元素 文本超出宽度时隐藏并用省略号代替
